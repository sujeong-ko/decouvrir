<h4 align="center">Decouvrir는 프랑스어로 "발견하다"라는 뜻으로,<br>
보다 쉽게 미술 작품을 올리고 구매하는 미술품 전문 거래 사이트입니다.</h4>

## **프로젝트 소개**


- 팀 명: 멈출 수 없는토끼🐰
- 프로젝트 기간: 2022.10.31 ~ 2022.11.12
- 기술 스택: HTML, CSS(Bulma), Javascript, Node.js, Express.js, AWS S3(이미지 업로드)

## **상세 기능**
### 유저
- 회원가입, 로그인
- 사용자 정보 조회
- 사용자 정보 수정
### 상품
- 전체 및 카테고리 별 작품 조회
- 작품 상세조회
- 작가별 작품 모아보기
- 작품 등록
### 구매
- 장바구니 담기, 조회
- 결제 정보 입력 후 구매
